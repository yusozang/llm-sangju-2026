**이시도르 지속가능연구소**(Isidor Sustainability Research Institute)가 주최하는 생성형 AI(LLM) 1일 집중 특강의 홍보·신청 단일 페이지 사이트입니다.

- **일시**: 2026년 8월 12일(수) 09:00–17:00
- **장소**: 상주 (세부 장소 추후 안내)
- **대상**: AI를 업무에 도입하고자 하시는 분 누구나
- **정원**: 선착순 15명 (정원 도달 시 자동 대기자 등록)
- **준비물**: 개인 노트북 + 충전기 반드시 지참 (종일 실습)
- **커리큘럼**: 오전 「AI 기본기 다지기」 · 오후 「내 업무 자동화하기」
- **문의**: isidor.yu@gmail.com

## 배포

GitHub Pages 정적 호스팅. `main` 브랜치 루트의 `index.html`을 서빙합니다.

## 구성

| 파일 | 역할 |
|------|------|
| `index.html` | 공개 랜딩 + 신청 폼 (단일 페이지) |
| `apps-script/Code.gs` | Google Apps Script 백엔드 — 구글시트 저장, 자동 이메일, 정원·대기자 관리 |
| `apps-script/Admin.html` | 관리자 페이지 (비밀번호 로그인, 현황·리스트, 참석확정 처리) |
| `SETUP-backend.md` | 백엔드 설치·배포 안내 |

## 설정 요약

1. `SETUP-backend.md`를 따라 Apps Script 프로젝트를 만들고 웹앱으로 배포합니다 (약 10분).
2. `index.html` 하단 `<script>`의 `API_URL`을 배포된 웹앱 주소(`.../exec`)로 교체 후 푸시합니다.
3. 관리자 페이지는 웹앱 URL 뒤에 `?page=admin`을 붙여 접속합니다 (비밀번호는 Script Properties의 `ADMIN_PASSWORD`).

`API_URL`이 플레이스홀더 상태이면 신청 버튼은 메일 클라이언트(mailto)로 대체 동작하고, 잔여석 배지는 표시되지 않습니다.

## 동작 흐름

신청 제출 → 상태 **신청접수** + 접수 확인 메일 자동 발송 → 관리자가 **참석확정** 클릭 → 확정 메일 자동 발송. 참석확정 인원이 15명에 도달하면 이후 신청자는 자동으로 **대기자**로 등록되고 대기 안내 메일을 받습니다.
